# IChing AstrBot Plugin

这是一个 AstrBot 插件，提供易经相关能力：

1. 今日运势
2. 数字起卦（每日限次）

## 功能说明

1. 今日运势
   - 根据北京时间日期生成当日干支与宜忌。
   - 基于日期哈希，同一天所有用户结果一致。

2. 数字起卦
   - 输入三个整数，返回上卦、下卦、变爻与卦辞概览。
   - 每个用户每天仅可成功起卦一次（北京时间 00:00 刷新）。

## 插件文件

1. `main.py`: AstrBot 插件入口（命令注册）。
2. `iching.py`: 易经核心逻辑。
3. `metadata.yaml`: AstrBot 插件元数据。

## AstrBot 命令

1. `今日运势`
2. `起卦 <数字1> <数字2> <数字3>`
3. `易经帮助`

示例：

```text
起卦 3 5 7
```

## 注意事项

1. 目前用户起卦记录保存在内存中，重启后会清空。
2. 代码内部固定使用北京时间（UTC+8）。
